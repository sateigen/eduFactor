# eduFactor

## Goal
  - Build a Django app to help students become more comfortable with web-based, interactive math questions.

## Live
  - Visit our live site: http://edufactor.herokuapp.com/

## Tools
  - Django, Django rest-framework
  - JavaScript, jQueryUI
